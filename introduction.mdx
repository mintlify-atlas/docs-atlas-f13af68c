## What is Wren LSP?

Wren LSP provides intelligent code editing features for Wren files across multiple editors. Written in Zig for speed and reliability, it implements the Language Server Protocol to deliver a consistent development experience regardless of your editor choice.

## Features

Wren LSP includes comprehensive support for modern code editing:

### Core navigation

- **Go to definition** - Jump to where symbols are defined
- **Find references** - Locate all uses of a symbol across your codebase
- **Workspace symbols** - Search for classes, methods, and variables by name
- **Document symbols** - View an outline of the current file structure

### Code intelligence

- **Completion** - Context-aware suggestions for identifiers, class members, built-in functions, and methods
- **Hover** - View type information and documentation on hover
- **Signature help** - See parameter information while calling functions
- **Inlay hints** - Type annotations displayed inline for variables

### Code quality

- **Diagnostics** - Syntax errors and semantic warnings as you type
- **Code actions** - Quick fixes for common issues
- **Rename** - Safely rename symbols throughout your project

### Editor integration

- **Semantic tokens** - Rich syntax highlighting based on semantic meaning
- **Document highlights** - Highlight related symbols in the current file
- **Folding ranges** - Collapse and expand code blocks
- **Selection range** - Smart selection expansion

<Note>
Most features work within a single file. Cross-module navigation (multi-file support) is under active development.
</Note>

## Example

Here's what Wren code looks like with LSP features:

```wren example.wren
class Logger {
  static info(message) {
    System.print("[INFO] " + message)
  }

  static error(message) {
    System.print("[ERROR] " + message)
  }
}

class MathUtils {
  static clamp(value, min, max) {
    if (value < min) return min
    if (value > max) return max
    return value
  }
}

// Type 'Logger.' to see completion suggestions
// Hover over 'clamp' to see its signature
// Use go-to-definition to jump to the class
Logger.info("Clamped: %(MathUtils.clamp(15, 0, 10))")
```

With Wren LSP active, you'll get:

- Autocompletion for `Logger.info` and `MathUtils.clamp`
- Parameter hints showing `(value, min, max)` when calling `clamp`
- Type information on hover
- Instant syntax error detection
- Quick navigation to symbol definitions
